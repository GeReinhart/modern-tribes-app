Option 4: Use Docker (Best for Production)

If you're deploying, use a Docker image that includes build tools:

Dockerfile:

dockerfile
FROM python:3.14-slim

# Install build dependencies
RUN apt-get update && apt-get install -y \
    gcc \
    python3-dev \
    && rm -rf /var/lib/apt/lists/*

WORKDIR /app
COPY requirements.txt .
RUN pip install --no-cache-dir -r requirements.txt

COPY . .
CMD ["uvicorn", "main:app", "--host", "0.0.0.0"]

Recommended Solution

Use Option 1 (install dev headers) for local development, and Option 4 (Docker) for production. This ensures asyncpg has full C extension support for best performance.
