## 2. рҹ”§ Remove Debug Settings

### **backend/app/config.py**
```python
from pydantic_settings import BaseSettings
from typing import Optional

class Settings(BaseSettings):
    # App
    APP_NAME: str = "YourApp"
    DEBUG: bool = False  # Set to False for production
    ENV: str = "production"

    # MongoDB
    MONGODB_URL: str
    MONGODB_DB_NAME: str

    # Security
    SECRET_KEY: str
    CORS_ORIGINS: list[str] = ["https://yourdomain.com"]

    # API
    API_V1_PREFIX: str = "/api/v1"

    class Config:
        env_file = ".env"
        case_sensitive = True

settings = Settings()
```

### **backend/app/main.py**
```python
from fastapi import FastAPI
from fastapi.middleware.cors import CORSMiddleware
from contextlib import asynccontextmanager
from motor.motor_asyncio import AsyncIOMotorClient
from app.config import settings

@asynccontextmanager
async def lifespan(app: FastAPI):
    # Startup
    app.mongodb_client = AsyncIOMotorClient(settings.MONGODB_URL)
    app.mongodb = app.mongodb_client[settings.MONGODB_DB_NAME]
    yield
    # Shutdown
    app.mongodb_client.close()

app = FastAPI(
    title=settings.APP_NAME,
    debug=settings.DEBUG,  # Uses settings
    docs_url="/api/docs" if settings.DEBUG else None,  # Disable in prod
    redoc_url="/api/redoc" if settings.DEBUG else None,
    lifespan=lifespan
)

# CORS
app.add_middleware(
    CORSMiddleware,
    allow_origins=settings.CORS_ORIGINS,
    allow_credentials=True,
    allow_methods=["*"],
    allow_headers=["*"],
)

@app.get("/health")
async def health_check():
    return {"status": "healthy"}
```

## 3. рҹҗі Docker Setup

### **backend/Dockerfile**
```dockerfile
FROM python:3.11-slim

WORKDIR /app

# Install dependencies
COPY requirements.txt .
RUN pip install --no-cache-dir -r requirements.txt

# Copy application
COPY . .

# Create non-root user
RUN useradd -m -u 1000 appuser && chown -R appuser:appuser /app
USER appuser

# Expose port
EXPOSE 8000

# Run application
CMD ["uvicorn", "app.main:app", "--host", "0.0.0.0", "--port", "8000"]
```

### **frontend/Dockerfile**
```dockerfile
# Build stage
FROM node:20-alpine as build

WORKDIR /app

COPY package*.json ./
RUN npm ci

COPY . .
RUN npm run build

# Production stage
FROM nginx:alpine

# Copy built assets
COPY --from=build /app/dist /usr/share/nginx/html

# Copy nginx config
COPY nginx.conf /etc/nginx/conf.d/default.conf

EXPOSE 80

CMD ["nginx", "-g", "daemon off;"]
```

### **frontend/nginx.conf**
```nginx
server {
    listen 80;
    server_name _;
    root /usr/share/nginx/html;
    index index.html;

    # Gzip compression
    gzip on;
    gzip_types text/plain text/css application/json application/javascript text/xml application/xml text/javascript;

    # Security headers
    add_header X-Frame-Options "SAMEORIGIN" always;
    add_header X-Content-Type-Options "nosniff" always;
    add_header X-XSS-Protection "1; mode=block" always;

    # API proxy
    location /api {
        proxy_pass http://backend:8000;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection 'upgrade';
        proxy_set_header Host $host;
        proxy_cache_bypass $http_upgrade;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
    }

    # SPA routing
    location / {
        try_files $uri $uri/ /index.html;
    }

    # Cache static assets
    location ~* \.(js|css|png|jpg|jpeg|gif|ico|svg|woff|woff2|ttf|eot)$ {
        expires 1y;
        add_header Cache-Control "public, immutable";
    }
}
```

### **docker-compose.prod.yml** (Production)
```yaml
version: '3.8'

services:
  mongodb:
    image: mongo:7
    container_name: app-mongodb
    restart: unless-stopped
    environment:
      MONGO_INITDB_ROOT_USERNAME: ${MONGO_USERNAME}
      MONGO_INITDB_ROOT_PASSWORD: ${MONGO_PASSWORD}
    volumes:
      - mongodb_data:/data/db
      - ./backups:/backups
    networks:
      - app-network
    command: mongod --auth

  backend:
    build:
      context: ./backend
      dockerfile: Dockerfile
    container_name: app-backend
    restart: unless-stopped
    environment:
      - DEBUG=false
      - ENV=production
      - MONGODB_URL=mongodb://${MONGO_USERNAME}:${MONGO_PASSWORD}@mongodb:27017
      - MONGODB_DB_NAME=${MONGO_DB_NAME}
      - SECRET_KEY=${SECRET_KEY}
      - CORS_ORIGINS=${CORS_ORIGINS}
    depends_on:
      - mongodb
    networks:
      - app-network

  frontend:
    build:
      context: ./frontend
      dockerfile: Dockerfile
    container_name: app-frontend
    restart: unless-stopped
    ports:
      - "80:80"
      - "443:443"
    depends_on:
      - backend
    networks:
      - app-network
    volumes:
      - ./ssl:/etc/nginx/ssl:ro  # For SSL certificates

volumes:
  mongodb_data:
    driver: local

networks:
  app-network:
    driver: bridge
```

## 7. рҹ’ҫ MongoDB Backup Commands

### **backup.sh**
```bash
#!/bin/bash

# Configuration
TIMESTAMP=$(date +%Y%m%d_%H%M%S)
BACKUP_DIR="./backups"
CONTAINER_NAME="app-mongodb"
DB_NAME="${MONGO_DB_NAME:-appdb}"
MONGO_USER="${MONGO_USERNAME:-admin}"
MONGO_PASS="${MONGO_PASSWORD}"

# Create backup directory
mkdir -p "$BACKUP_DIR"

# Backup
echo "Starting backup of $DB_NAME..."
docker exec $CONTAINER_NAME mongodump \
  --username="$MONGO_USER" \
  --password="$MONGO_PASS" \
  --db="$DB_NAME" \
  --authenticationDatabase=admin \
  --out="/backups/backup_$TIMESTAMP"

# Compress
echo "Compressing backup..."
cd "$BACKUP_DIR"
tar -czf "backup_${TIMESTAMP}.tar.gz" "backup_$TIMESTAMP"
rm -rf "backup_$TIMESTAMP"

# Keep only last 7 backups
ls -t backup_*.tar.gz | tail -n +8 | xargs -r rm

echo "Backup completed: backup_${TIMESTAMP}.tar.gz"
```

### **restore.sh**
```bash
#!/bin/bash

# Configuration
BACKUP_FILE=$1
CONTAINER_NAME="app-mongodb"
DB_NAME="${MONGO_DB_NAME:-appdb}"
MONGO_USER="${MONGO_USERNAME:-admin}"
MONGO_PASS="${MONGO_PASSWORD}"

if [ -z "$BACKUP_FILE" ]; then
    echo "Usage: ./restore.sh <backup_file.tar.gz>"
    exit 1
fi

# Extract backup
echo "Extracting backup..."
BACKUP_DIR=$(basename "$BACKUP_FILE" .tar.gz)
tar -xzf "$BACKUP_FILE" -C "./backups/"

# Restore
echo "Restoring $DB_NAME..."
docker exec $CONTAINER_NAME mongorestore \
  --username="$MONGO_USER" \
  --password="$MONGO_PASS" \
  --db="$DB_NAME" \
  --authenticationDatabase=admin \
  "/backups/$BACKUP_DIR/$DB_NAME"

# Cleanup
rm -rf "./backups/$BACKUP_DIR"

echo "Restore completed!"
```

### **Make scripts executable**
```bash
chmod +x backup.sh restore.sh
```

### **Manual Backup Commands**
```bash
# Quick backup
docker exec app-mongodb mongodump \
  --username=admin \
  --password=yourpassword \
  --authenticationDatabase=admin \
  --out=/backups/manual_backup

# Export specific collection to JSON
docker exec app-mongodb mongoexport \
  --username=admin \
  --password=yourpassword \
  --authenticationDatabase=admin \
  --db=appdb \
  --collection=users \
  --out=/backups/users.json

# Restore from backup
docker exec app-mongodb mongorestore \
  --username=admin \
  --password=yourpassword \
  --authenticationDatabase=admin \
  /backups/manual_backup

# Copy backup from container to host
docker cp app-mongodb:/backups/manual_backup ./backups/
```

### **Automated Backups with Cron**
```bash
# Edit crontab
crontab -e

# Add daily backup at 2 AM
0 2 * * * cd /path/to/your-app && ./backup.sh >> ./backups/backup.log 2>&1
```
